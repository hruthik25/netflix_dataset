# Netflix Dataset

This repository contains a Netflix dataset, which includes details about shows such as ID, type, title, director, country, date added, release year, rating, duration, and genre.

## Data Description

The dataset `netflix_data.csv` contains the following columns:
- `show_id`: Unique ID for each show
- `type`: Type of the show (e.g., Movie, TV Show)
- `title`: Title of the show
- `director`: Director(s) of the show
- `country`: Country where the show was produced
- `date_added`: Date the show was added to Netflix
- `release_year`: Year the show was originally released
- `rating`: Rating of the show
- `duration`: Duration of the show
- `listed_in`: Categories the show is listed under

## Project Structure

- `netflix_data.csv`: Main dataset file containing Netflix shows.
- `Netflix_Analysis.ipynb`: Jupyter notebook with the analysis of the dataset.
